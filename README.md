# Vision-Based Navigation System

Real-time navigation using Computer Vision and YOLOv3-tiny.

## Features
- Edge detection + ROI
- 5-sector navigation logic
- Temporal smoothing
- YOLO object detection
- Proximity-aware decisions

## Structure
src/ → code  
yolo/ → config files  
data/ → input video  

## Setup
Download YOLO weights manually and place in /yolo
